# ⚡ HTMX Task Manager


A modern CRUD application demonstrating
Server Driven UI using HTMX.


## 🚀 Tech Stack


Frontend:

- HTML
- CSS
- HTMX


Backend:

- Node.js
- Express.js



## Features


✅ Create Tasks

✅ View Tasks

✅ Edit Tasks

✅ Delete Tasks

✅ Search

✅ Partial Page Updates

✅ No React

✅ No Vue

✅ No Angular



## Architecture


Traditional SPA:


Browser

↓

JavaScript

↓

API JSON

↓

Update DOM



HTMX:


Browser

↓

HTTP Request

↓

Server Generates HTML

↓

HTMX Swaps Fragment



## Run Project



Install:


npm install



Start:


npm start



Open:


http://localhost:3000



## Why HTMX?


HTMX keeps the simplicity of traditional
server rendering while adding modern
interactive behaviour.
